# BatchFrame

BatchFrame is a browser-based image framing tool for batch cropping, signing, adding borders, and exporting square image sets.

## Features

- Batch load JPG, PNG, HEIC, HEIF, and DNG image files.
- Generate four export files per image with consistent timestamp-based filenames.
- Add a signature mark and choose its position.
- Adjust the master composition before export.
- Export a single image set or a ZIP of all loaded sets.

## Deployment

This is a static web app. It can be hosted with GitHub Pages by serving the repository root.
